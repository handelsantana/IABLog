## IABlog — Portal de Notícias sobre Inteligência Artificial

Projeto completo de um blog/portal de notícias sobre IA com Next.js (App Router), TypeScript e Material UI.

## Rodando o projeto

Instale as dependências:

```bash
npm install
```

Inicie o ambiente de desenvolvimento:

```bash
npm run dev
```

Para build de produção:

```bash
npm run build
npm start
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Estrutura principal

- `app/` — rotas e páginas (home, notícia, tag, busca, 404)
- `components/` — UI reutilizável com MUI
- `lib/` — dados mock e helpers
- `public/images/` — capas das notícias

## Stack

- Next.js App Router + TypeScript
- Material UI v5
- Dados mock locais
- Tema claro/escuro com toggle

---

Feito para ser rápido, acessível e 100% responsivo.
